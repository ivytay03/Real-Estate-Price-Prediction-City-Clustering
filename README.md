## 🏠 Real Estate Price Prediction & City Clustering

### 📌 Overview
This project applies data analysis techniques to predict property prices and segment cities based on lifestyle characteristics using clustering methods.

---

### 🎯 Objective
- To predict housing prices based on key features  
- To segment cities into lifestyle groups  
- To generate insights and recommendations from data  

---

### 📊 Dataset
- Real estate dataset (housing features such as size, bedrooms, location, distance)  
- City lifestyle dataset (income, rent, and other indicators)  

---

###  Methodology

#### 🔹 Price Prediction
- Used regression techniques to predict property prices  
- Considered features such as:
  - Square footage  
  - Number of bedrooms  
  - Location score  
  - Distance to city center  

#### 🔹 Clustering Analysis
- Applied clustering to group cities into distinct lifestyle segments  
- Identified patterns based on income and rental indicators  

---

### 📈 Key Insights
- Property prices are strongly influenced by location and size  
- Cities can be grouped into different lifestyle segments based on income and rent  
- Clustering helps identify target areas for investment or living  

---

### 🛠 Tools Used
- R (Data Analysis & Visualization)  
- Clustering Techniques  
- Regression Analysis  

---

### 🖼 Output
- Price prediction results  
- Cluster visualization of cities
- Recommendation
